# 生产环境部署检查清单

使用此清单确保生产环境部署安全可靠。

---

## 部署前检查

### 基础设施

- [ ] 服务器资源满足最低要求（2核4G+）
- [ ] 使用 SSD 存储（推荐）
- [ ] 操作系统已更新到最新安全补丁
- [ ] 有足够的磁盘空间（建议 20GB+）
- [ ] 有备份和恢复计划

### 网络配置

- [ ] 安全组/防火墙规则已正确配置
- [ ] 只开放必要的端口（22, 80, 443, 6116）
- [ ] 域名已配置（如使用）
- [ ] DNS 解析正常

### 安全准备

- [ ] 已创建非 root 用户运行服务
- [ ] SSH 已禁用 root 登录
- [ ] SSH 使用密钥认证，禁用密码登录
- [ ] 防火墙已配置并启用
- [ ] Fail2ban 或类似防护已安装（可选但推荐）

---

## 部署中检查

### 配置文件

- [ ] `JWT_SECRET` 已修改为强随机密钥
- [ ] 所有默认密码已修改
- [ ] 敏感信息不提交到版本控制
- [ ] `.env` 文件权限设置为 600
- [ ] 数据库文件权限设置正确

### Docker 配置

- [ ] 数据卷已正确配置持久化
- [ ] 资源限制已设置（可选）
- [ ] 健康检查已配置
- [ ] 重启策略已设置

### MQTT 配置

- [ ] MQTT 连接使用 TLS/SSL
- [ ] MQTT 用户名密码已配置
- [ ] 可以正常连接到 EMQX 服务器

---

## 部署后检查

### 功能验证

- [ ] 可以访问 IoT Manager 界面
- [ ] 可以注册和登录用户
- [ ] 可以添加和管理设备
- [ ] 可以发送控制命令
- [ ] 设备状态可以正常显示

### 监控验证

- [ ] Prometheus 可以正常访问
- [ ] Grafana 可以正常登录
- [ ] 监控仪表板数据正常显示
- [ ] 指标采集正常

### 安全检查

- [ ] HTTPS 已启用（如适用）
- [ ] 安全响应头正常（X-Frame-Options 等）
- [ ] 速率限制正常工作
- [ ] 没有敏感信息泄露

### 备份检查

- [ ] 自动备份已启用（如配置）
- [ ] 备份文件正确生成
- [ ] 备份文件权限正确
- [ ] 测试过从备份恢复流程

---

## 日常维护检查

### 每日检查

- [ ] 检查服务器资源使用率（CPU、内存、磁盘）
- [ ] 查看容器运行状态
- [ ] 检查是否有错误日志

### 每周检查

- [ ] 验证备份正常工作
- [ ] 检查系统更新
- [ ] 审查安全日志

### 每月检查

- [ ] 更新 Docker 镜像
- [ ] 更新系统包
- [ ] 执行安全扫描
- [ ] 清理旧日志和备份

---

## 紧急情况准备

- [ ] 有书面的故障恢复流程
- [ ] 备份定期测试
- [ ] 关键配置有文档记录
- [ ] 有联系方式支持

---

## 性能优化（可选）

- [ ] 启用 HTTPS 缓存
- [ ] 配置 CDN（如需要）
- [ ] 优化数据库查询
- [ ] 配置 Prometheus 数据保留策略
